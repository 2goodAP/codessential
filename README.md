# Codessential
Currently just a simple WebIDE :)

### Running the editor
In order to run the project, please follow the following steps:

+ Obtain dependencies by running:
        
        $ npm install
        
+ Then run
        
        $ npm run build
        
  to obtain the 'dist' directory
+ *Optionally*, to start a live server, run
        
        $ npm start
